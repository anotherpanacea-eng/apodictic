# Register / Stance Pilot Expected Profiles

Severity vocabulary is canonical: Must-Fix / Should-Fix / Could-Fix.

## E7 — good-faith hybrid

- Intake: writer-confirmed `register=generative`; high-stakes gate inactive.
- Journey: AT5 coherence/fertility succeeds. Lens-method WR/CL findings, if returned,
  remain Could-Fix with `calibration_effect=register-floor` and no `cash_out_ref`.
- Landing: exactly one hard prescriptive cash-out, joined as `CO1`. Any finding there
  records `calibration_effect=blocked-cash-out` and stays at asserted burden.
- Required landing pressure: policy comparison, evidence, threshold, administrative-
  discretion, and implementation-cost questions remain live. No register halo.
- No document-wide re-flood: journey findings do not rise merely because CO1 is high
  consequence.

Ship evidence requires two independent blind runs that agree on those five properties.
