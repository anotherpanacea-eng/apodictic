# Coaching History — Example

*Rolling, opt-in, local-only. Each observation is mechanically derived from your recorded session
history (a count over the Findings-Ledger dispositions and the revision-arc phases), then written as
what the coach noticed — not a score. The counts are the coach's evidence; what you read is an open
question for your reading. This is a non-governed project, so the session-to-session history behind a
recurrence comes from the coach's own running notes, not an independently checked event log — the
observations below say so plainly. This record is yours: run `coaching_history.py delete .` to remove
it (that also revokes consent to re-derive it).*
